# Imobiliária Angular

Aplicação web desenvolvida em Angular com o objetivo de simular uma plataforma imobiliária para exibição e gerenciamento de imóveis. O projeto foi construído durante os estudos do framework Angular, acompanhando um curso disponibilizado pela própria equipe do Angular no YouTube.

## Sobre o Projeto

O Imobiliária Angular é uma aplicação front-end que simula o funcionamento de um portal imobiliário, permitindo a visualização de imóveis e a navegação entre diferentes páginas da aplicação.

O projeto foi desenvolvido com foco no aprendizado das principais funcionalidades do Angular, aplicando conceitos modernos de desenvolvimento de interfaces e organização de aplicações SPA (Single Page Application).

## Objetivos do Projeto

* Aprender a estrutura de uma aplicação Angular.
* Trabalhar com componentes reutilizáveis.
* Praticar roteamento entre páginas.
* Aplicar conceitos de Data Binding.
* Utilizar serviços e organização em módulos.
* Desenvolver interfaces responsivas.

## Funcionalidades

* Listagem de imóveis.
* Exibição de detalhes dos imóveis.
* Navegação entre páginas.
* Componentes reutilizáveis.
* Interface responsiva.
* Organização dos dados em componentes e serviços.

## Tecnologias Utilizadas

* Angular
* TypeScript
* HTML5
* CSS3
* Angular CLI
* RxJS
* Node.js
* npm

## Arquitetura

A aplicação utiliza a arquitetura baseada em componentes do Angular, proporcionando:

* Separação de responsabilidades.
* Reutilização de código.
* Facilidade de manutenção.
* Organização da interface em componentes independentes.

## Aprendizados

Durante o desenvolvimento deste projeto foram praticados conceitos importantes do Angular, como:

* Componentes.
* Templates.
* Diretivas.
* Data Binding.
* Roteamento.
* Serviços.
* Organização de aplicações SPA.

## Execução do Projeto

